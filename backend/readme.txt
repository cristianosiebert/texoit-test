Backend feito em Python - API de Filmes

Como instalar as dependências:
Na raiz do projeto (dentro da pasta backend) execute no terminal ou no ambiente virtual: pip install -r requirements.txt
Se alguma instalação não ocorrer de forma satisfatória, repita o comando com permissão de admin/root.

Uma vez tudo instalado, na raiz do projeto (dentro da pasta backend), execute:
python server.py

Para executar os testes basta ir até a pasta raiz do projeto (dentro da pasta backend) e executar:
python -m pytest
